Hier staat hoe je de database importeerd om de php crud opdracht te laten werken.

Stap 1:
Start xampp/wampp of een andere web server client.

Stap 2: 
ga naar localhost/phpmyadmin

Stap 3:
Maak een nieuwe database aan en ga naar importeren.

Stap 4:
In de PHP folder staat een folder genaamd "SQL database voor CRUD", het sql bestand in deze map moet je importeren in phpmyadmin.

Stap 5: 
Druk op go of starten en je bent klaar!

daarna dien je alleen nog maar de php crud folder in de root van de webserver te slepen en daarna kan je bij localhost/.../PHP_CRUD de site zien!
